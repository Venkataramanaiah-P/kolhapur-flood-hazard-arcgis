# Kolhapur Flood Hazard Map — ArcGIS Pro

Multi-criteria weighted overlay flood hazard analysis 
for Kolhapur District, Maharashtra using ArcGIS Pro.

## Weights
| Factor | Weight |
|--------|--------|
| DEM (Elevation) | 30% |
| Flood SAR | 25% |
| Population | 20% |
| Slope | 15% |
| NDVI | 10% |

## Tools
- ArcGIS Pro Spatial Analyst
- Python Map Algebra (arcpy)
- EPSG:32643 UTM Zone 43N
